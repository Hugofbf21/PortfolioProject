A PERSONALIZED CHESS BOARD AND GAME
In the end of the "Programming Fundamentals" class in my first year i developed a fully customizable chess game in which
you could choose the dimensions of the board

You could also create boards without any borders and legends.
The game UI was done in the console and the
code written in the programming language Kotlin.

The game would also detect a win and keep track of the
wins of each player with personalized names asked
when the code first ran.
